# рҹ§  Recommendation Logic

The recommendation engine filters destinations based on three user-selected parameters:

- Budget
- Trip Duration
- Travel Interest

The application compares these preferences against a predefined dataset and displays only the matching destinations.

If no destination satisfies the selected criteria, the application informs the user with a friendly message.
